Class LocusTree::Tree
In: lib/locus_tree.rb
Parent: Object

Description

The LocusTree::Tree class is similar to an R-Tree. It contains a collection of nodes where the toplevel node covers the whole of a chromosome and the bottom level nodes (level 0) correspond to single raw datapoints. Different chromosomes cannot be part of the same tree (see LocusTree::Container).
